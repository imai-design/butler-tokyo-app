# BUTLER TOKYO をiPhoneのアプリにする（非公式）

高級デリバリー [BUTLER TOKYO](https://butler-tokyo.com/) の注文サイトを、iPhoneのホーム画面にアプリとして追加するための非公式の個人用ガイドです。BUTLER TOKYO公式とは関係ありません。名称・ロゴは各権利者に帰属します。

ガイド: https://imai-design.github.io/butler-tokyo-app/

- `index.html` — インストールガイド（方法01=構成プロファイル / 方法02=Safariのホーム画面に追加）
- `butler-tokyo.mobileconfig` — Web Clip構成プロファイル（ホーム画面にアイコンを置くだけ。データ収集・端末管理なし・削除自由）
